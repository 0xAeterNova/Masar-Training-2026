# Session 2 - Introduction to Linux Operating System Part 1

This directory contains a brief overview of **Session 2: Introduction to Linux Operating System Part 1** from **Masar Training 2026**.

## Overview

This session introduces the Linux operating system and builds the foundation needed to work comfortably in a Linux environment. It covers the basic concepts of Linux, the setup of Kali Linux, command-line interaction, file system navigation, essential commands, permissions, and user management.

## Main Topics

- Introduction to Linux and Unix-like operating systems
- Kali Linux setup using virtualization
- Terminal and shell basics
- Linux file system structure
- Navigating directories and paths
- Basic Linux commands
- Files and permissions
- Linux user management

## Key Learning Areas

### Linux Fundamentals
The session explains what Linux is, how it relates to Unix and Unix-like systems, and why it is widely used in technical and cybersecurity environments.

### Working in the Command Line
It introduces the terminal and shell, along with essential commands such as `ls`, `cd`, `pwd`, `cp`, `mv`, `touch`, `mkdir`, `cat`, `grep`, and output redirection.

### File System and Permissions
The session covers the Linux directory structure, file navigation, standard permissions, and advanced permissions such as **SUID**, **SGID**, and the **sticky bit**.

### User Management
Basic user administration is introduced through system files like `/etc/passwd`, `/etc/shadow`, and `/etc/group`, in addition to common user management commands.

## Purpose of This Session

This session provides the practical Linux foundation required for later technical sessions in the program, especially those related to security, system usage, and hands-on lab work.

## Note

Understanding the topics in this session is important because Linux is a core environment used throughout many cybersecurity tasks and tools.
