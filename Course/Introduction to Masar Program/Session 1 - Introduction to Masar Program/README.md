# Session 1 - Introduction to Masar Program

This directory contains a brief overview of **Session 1: Introduction to Masar Program** from **Masar Training 2026**.

## Overview

This session introduces the Masar program, the core areas of cybersecurity, and the main learning path followed throughout the training. It gives a high-level understanding of how the program is structured and what participants are expected to learn during the coming weeks.

## Main Topics

The session focuses on:

- Introduction to cybersecurity
- Cybersecurity roles
- Program outline and learning path
- Program prerequisites
- Training structure and schedule
- Grading and assessment
- Program platforms
- Attendance policy and code of conduct

## Key Learning Areas

### Introduction to Cybersecurity
The session explains cybersecurity as the practice of protecting digital assets from unauthorized access, disruption, or destruction. It also introduces common threat actors and the importance of both proactive and defensive security approaches.

### Cybersecurity Roles
A clear overview is provided for the three major cybersecurity domains covered in the program:

- **Offensive Security**
- **Defensive Security**
- **Governance, Risk, and Compliance (GRC)**

The session also introduces example roles in each domain, helping build an early understanding of the different career paths in cybersecurity.

### Program Structure
The Masar program is presented as an 8-week training path that combines theory, practical exercises, and take-home assignments. It is designed to give participants both technical knowledge and hands-on experience across multiple cybersecurity areas.

## Purpose of This Session

This session sets the foundation for the rest of the program. It helps participants understand the training journey, the expectations, and the broader cybersecurity landscape before moving into the technical sessions.

## Note

This session is an important starting point because it connects the program goals, weekly structure, and cybersecurity domains into one clear roadmap for the learner.
