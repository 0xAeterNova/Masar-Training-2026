# Introduction to Masar Program

This directory contains the introductory sessions for **Masar Training 2026**.

## Overview

The **Introduction to Masar Program** section provides the starting point of the training and sets the foundation for the sessions that follow. It introduces the program structure, learning path, and the core Linux knowledge needed for the practical and technical parts of the course.

## Included Sessions

- **Session 1:** Introduction to Masar Program
- **Session 2:** Introduction to Linux Operating System Part 1
- **Session 3:** Introduction to Linux Operating System Part 2

## Purpose

This section is designed to help build a clear understanding of the program journey while preparing participants with the essential Linux fundamentals required throughout the training.

## Note

These sessions serve as the entry point to the program and create the baseline knowledge needed before moving into the more specialized cybersecurity tracks.
