# Session 3 - Introduction to Linux Operating System Part 2

This directory contains a brief overview of **Session 3: Introduction to Linux Operating System Part 2** from **Masar Training 2026**.

## Overview

This session builds on the Linux fundamentals introduced earlier and focuses on important administration and automation concepts used in daily Linux work. It covers sudo privileges, package management, services, processes, cron jobs, local DNS, shell types, and the basics of Bash scripting.

## Main Topics

- Linux sudo privileges
- Linux package management
- Linux services
- Linux processes
- Linux cron jobs
- Local DNS
- Linux shell types
- Bash scripting

## Key Learning Areas

### Sudo Privileges
The session explains how `sudo` works, why it is used, and how permissions are defined in the `/etc/sudoers` file. It also introduces examples of privilege rules and common commands such as `sudo -l`.

### Package Management
It introduces Linux package management concepts, including package installation, updates, and removal using tools such as `apt-get` and `yum`.

### Services and Processes
The session covers the difference between services and processes, how services are managed using `systemctl`, and how processes can be viewed, monitored, prioritized, and terminated.

### Cron Jobs and Local DNS
It explains how scheduled tasks are configured with cron jobs and how local name resolution works through the `/etc/hosts` file.

### Shells and Bash Scripting
The session introduces common Linux shell types and the foundations of Bash scripting, including variables, conditions, loops, functions, command substitution, and environment variables.

## Purpose of This Session

This session strengthens the practical Linux knowledge needed for system administration, automation, and later cybersecurity labs. It helps build confidence in managing Linux systems beyond basic command-line usage.

## Note

The topics in this session are especially useful because they connect Linux usage with real administrative and scripting tasks that appear frequently in technical and security environments.
