# Defensive Security

This directory contains the **Defensive Security** sessions for **Masar Training 2026**.

## Overview

The Defensive Security track focuses on protecting systems, detecting threats, responding to incidents, and strengthening security operations. It introduces the core blue team concepts and practical areas needed to monitor, investigate, and defend modern environments.

## Included Sessions

- **Session 13:** Introduction to Blue Team
- **Session 14:** Incident Handling/Response
- **Session 15:** Incident Handling/Response Part Two
- **Session 16:** Digital Forensics
- **Session 17:** Digital Forensics Part Two
- **Session 18:** Secure Configuration Management (SCM)
- **Session 19:** Secure Configuration Management (SCM) Part Two
- **Session 20:** Threat Intelligence and Threat Hunting
- **Session 21:** Threat Intelligence and Threat Hunting Part 2

## Purpose

This section is designed to build a solid foundation in defensive security by covering the main areas of detection, response, investigation, hardening, and proactive threat analysis.

## Note

These sessions provide the essential defensive knowledge needed to understand how organizations monitor threats, respond to incidents, and improve their overall security posture.
