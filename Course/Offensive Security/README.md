# Offensive Security

This directory contains the **Offensive Security** sessions for **Masar Training 2026**.

## Overview

The Offensive Security track focuses on the practical and technical foundations of identifying, analyzing, and understanding security weaknesses in systems, applications, and environments. It introduces the core concepts, methodologies, and tools commonly used in offensive security work.

## Included Sessions

- **Session 4:** Introduction to Ethical Hacking Methodologies
- **Session 5:** Reconnaissance & Scanning
- **Session 6:** Vulnerability Assessment
- **Session 7:** Exploitation
- **Session 8:** Linux Post Exploitation
- **Session 9:** Clean up, Reporting & C2
- **Session 10:** Secure Coding Foundations
- **Session 11:** Application Security Vulnerabilities
- **Session 12:** DevSecOps

## Purpose

This section is designed to build a solid foundation in offensive security by covering the main stages of assessment, exploitation, post-exploitation, and secure development practices.

## Note

These sessions provide the core offensive security knowledge needed to understand how security weaknesses are discovered, analyzed, and addressed in real-world environments.
