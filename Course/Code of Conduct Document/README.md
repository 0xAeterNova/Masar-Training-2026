# Code of Conduct

This directory contains the **Masar Training Code of Conduct** document for the **Masar Training 2026** program.

## Overview

The Code of Conduct defines the standards, policies, and expectations that guide participation throughout the program. It is intended to maintain a professional, respectful, and secure learning environment for all participants.

## What It Covers

The document outlines the main expectations related to:

- Attendance and punctuality
- Engagement and participation
- Assignments and deadlines
- Professionalism and behavior
- Use of tools and platforms
- Security, ethics, and social media
- Communication policy
- Recording and privacy
- Consequences of violations

## Purpose

This document helps ensure that all participants understand their responsibilities and follow a consistent standard of conduct during the training. It also supports a safe and productive environment aligned with the goals of the Masar program.

## Note

All participants are expected to read, understand, and comply with the Code of Conduct throughout the training period.
